# 用户相关

## 用户登录

```
POST /user/signin
```

参数

```
{
    username: "fawaikuangtu",
    password: "zhangsan"
}
```

返回

```
{
    code: 100,
    msg: "成功",
    data: {
        ?????token
    }
}

{
    code: 400,
    msg: "用户名不存在",
    data: null
}
{
    code: 400,
    msg: "密码错误",
    data: null
}
```

    edu_bg: "幼儿园",
    photo: url,

```

```

    code: 100,
